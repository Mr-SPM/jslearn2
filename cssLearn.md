# CSS学习
1. 选择器
+ id 选择器
```
<div id='test' >id 选择器 </div>
#test {font-size:18px;}
```
+ class 选择器
```
<div class='test' >class 选择器 </div>
.test {font-size:18px;}
或者  指定特定的HTML元素使用class
div..test {font-size:18px;}
```
